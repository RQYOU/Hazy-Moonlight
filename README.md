# Hazy Moonlight：A Theme for SiYuan

## Introduction (介绍)

A light theme for [SiYuan](http://github.com/siyuan-note/siyuan).

[思源笔记](http://github.com/siyuan-note/siyuan)的一款浅色主题。整体采用融为一体的、朦胧、清爽的风格。
